## [0.0.1] - 2026-07-10

### Features

- Initial Dacely CLI: a themed Ink + React terminal interface with an
  interactive splash screen and REPL (slash commands such as `/help` and
  `/login`), plus `dacely <command>` and `dacely --flag` argument parsing.
  Commands ship as placeholders behind a clean object-oriented core.
